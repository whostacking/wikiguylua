## Installation

This bot runs on the [Luvit](https://luvit.io) runtime.

1. Install Luvit and Lit:
   ```bash
   curl -L https://github.com/luvit/lit/raw/master/get-lit.sh | sh
   ```
2. Install dependencies:
   ```bash
   ./lit install
   ```
3. Run the bot:
   ```bash
   ./luvit main.lua
   ```
